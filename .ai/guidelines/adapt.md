### Clone & Adapt Protocol

- Clone the closest existing backend route/controller/service/schema pattern or frontend feature/component/table/form pattern before inventing a new one.
- Keep module structure consistent across students, teachers, courses, exams, attendance, payments, and settings.
- Change resource-specific names, schemas, routes, columns, labels, and API calls only.
- Extract shared behavior instead of copy-pasting it a third time.
