# Domain Glossary

OCMS covers campus administration, academics, attendance, results, finance, and settings.

## Backend-Backed Models

Current Prisma models:

- `Role`
- `User`
- `Department`
- `Student`
- `Course`
- `ExamSchedule`
- `ExamResult`
- `CourseExam`

## Frontend Modules To Verify

The frontend also has screens for teachers, attendance, payments, invoices, settings, permissions, and results. Before treating those as complete features, verify backend API/model support.
